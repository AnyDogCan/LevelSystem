- Через asset menu Create->LevelSystem создаём в рес-ах LevelSystemSettings (должен быть 1) и Levels

- Levels заполняем нужными сценами в порядке следования (сцены не забываем добавить в buid settings)

- В LevelSystemSettings добавляем ссылку на Levels и выбираем нужный тип зацикливания уровней. Также указываем стартовую
сцену. Если мы запустим игру не со стартовой сцены, то включется дебаг мод. В дебаг моде работает только релоад сцены.


- Работаем через статический класс LevelProcessor.
    Св-ва:
        CurrentIndex - индекс текущего уровня
        LevelActions - возвращает интерфейс ILevelActions (см. ниже)
    Методы:
        - RequestNextLevel() - запрашиваем загрузку след уровня
        - ReloadLevel() - запрашиваем перезагрузку текущего уровня
        - LoadPreviousLevel() - загружаем предыдущий уровень
        - CompleteLevel() - объявляем об успещном завершении уровня

ILevelActions служит для отслеживания событий прогресса уровня.
    Определённые события:
        Action<int> LevelLoaded - уровень загружен
        Action LevelCompleted - уровень пройден
        Action NextLevelRequested - запрошен следующий уровень


Сразу же будет загружена сцена согласно твоему прогрессу (не важно с какой сцены стартовать)
